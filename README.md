# Porfolio_Project101

Hi! I'm Hannah, and welcome to my portfolio! This project shows my skills, sample projects, and resume as I start my journey as an IT professional. The site is designed using HTML, CSS, and JavaScript with a clean and modern look.

How to run locally :

1. Go to this GitHub repository.  
2. Click on the **“Code”** button (green button at the top right).  
3. Choose **“Download ZIP”** and extract it on your computer.  
4. Open the folder and double-click **`index.html`** to open it in your browser.
